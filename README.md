## Features

- Read 10 websites get logo and phone numbers
- Regex Phone Number
- BeautifulSoup crawler
- Async method non-block

## Installation
Main
```sh
cd project/
pip install -r requirements.txt
python3 main.py
```

Docker
```sh
docker build -t crawler-phone-logo .
docker run -it crawler-phone-logo
```
